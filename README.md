# 🚀 duifene-download (对分易批量下载助手)

![Version](https://img.shields.io/badge/version-2.6-blue)
![License](https://img.shields.io/github/license/HMuSeaB/duifene-download)

这是一个用于**对分易 (Duifene)** 教学平台的油猴脚本。它主要用于解锁受限资源下载，并提供舒适的 UI 和批量操作功能。

## ✨ 主要功能 (V2.6)

* **🔒 自动解锁**：自动检测被禁止下载的文件，恢复下载按钮。
* **🎨 UI 优化**：
    * 复选框**绝对定位**，不再挤压文件图标，界面更整洁。
    * 新增可**拖拽悬浮面板**，位置自动记忆。
* **⚡ 批量操作**：
    * 支持**“本页全选 / 取消选择”**智能切换。
    * 提供**批量下载**功能（自动逐个弹出下载窗口，无需手动点击）。
* **🔧 极致轻量**：纯原生 JS 实现，不依赖任何第三方库（如 JSZip），加载速度极快，兼容性最好。

## 📥 安装方法

1.  首先安装浏览器扩展 **Tampermonkey (油猴)**。
2.  **[点击这里安装脚本](https://github.com/你的GitHub用户名/duifene-download/raw/main/duifene-helper.user.js)**
    *(注意：上传代码后，点击 duifene-helper.user.js 文件页面的 "Raw" 按钮获取真实链接替换此处)*

## 🛠️ 使用说明

1.  进入对分易的“课程资源”页面。
2.  右下角会自动出现悬浮管理面板（也可以手动勾选文件）。
3.  点击 **“本页全选”**。
4.  点击 **“立即批量下载”**，浏览器会依次弹出保存框。

## ⚠️ 注意事项

* 批量下载时，浏览器可能会拦截弹窗。请在地址栏右侧**允许该网站的弹窗和重定向**。
* 本脚本仅供学习交流，请勿用于商业用途。
